1. Support contact

  • Email: shteosis@gmail.com
  • Repository: Project repository / 프로젝트 저장소
  • Issue tracker: Issue tracker / 이슈 트래커
  • Discussions: Discussions / 디스커션

Email is the primary direct support path for the current public build.

2. What LoRAbit support covers

  • app launch issues
  • workspace selection and file access issues
  • model inspection, preflight, dataset preparation, training, and export issues
  • questions about the current public build of the app itself

3. What support does not cover

  • granting rights to third-party models, datasets, or documents
  • changing the license or usage terms imposed by third-party model or dataset providers
  • guaranteeing compatibility for every local model layout

4. What to include in a support request

When possible, include the following:

  • compatibility report
  • preflight summary
  • training overview
  • run log
  • operating system version
  • device model
  • selected model input type, such as an MLX-ready directory or a safetensors-based model folder
  • whether the active workspace is the default workspace or a custom workspace
  • the exact warning or error shown on screen

5. Privacy

Privacy information for the current public build is available at privacy.html.

2. LoRAbit 지원 범위

  • 앱 실행 문제
  • 워크스페이스 선택 및 파일 접근 문제
  • 모델 검사, preflight, 데이터셋 준비, 학습, 내보내기 문제
  • 현재 공개 빌드 기준 앱 자체 동작에 대한 질문

3. 지원 범위에 포함되지 않는 항목

  • 제3자 모델, 데이터셋, 문서에 대한 권리 부여
  • 제3자 모델 또는 데이터셋 제공자의 라이선스나 사용 조건 변경
  • 모든 로컬 모델 레이아웃에 대한 호환성 보장

4. 문의 시 함께 보내면 좋은 정보

가능하면 아래 정보를 함께 보내 주세요.

  • compatibility report
  • preflight summary
  • training overview
  • run log
  • 운영체제 버전
  • 사용 중인 기기 모델
  • 선택한 모델 입력 형식 (예: MLX 준비 디렉터리 또는 safetensors 기반 모델 폴더)
  • 현재 워크스페이스가 기본 워크스페이스인지 사용자 지정 워크스페이스인지
  • 화면에 표시된 정확한 경고 또는 오류 문구

5. 개인정보처리방침
