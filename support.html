English

1. Public support paths

  • Project repository: https://github.com/Facta-Leopard/LoRabbit
  • Support email: shteosis@gmail.com
  • Issue tracker: https://github.com/Facta-Leopard/LoRabbit/issues
  • Discussions: https://github.com/Facta-Leopard/LoRabbit/discussions
  • Privacy policy: privacy.html

If Issues or Discussions are enabled in the repository, those are the primary public support paths for the current public build.

2. Before requesting support

When possible, include the following:

  • compatibility report
  • preflight summary
  • training overview
  • run log
  • macOS version
  • Apple Silicon device model
  • selected model format, such as MLX-ready directory, safetensors-based directory, or GGUF source
  • whether the active workspace is the default workspace or a custom workspace

This helps distinguish model-format issues, workspace-access issues, dataset-preprocessing issues, and training-memory issues.

3. What to describe in a support request

  • what you selected as the model input
  • what you selected as the dataset input
  • whether the issue appears during model inspection, dataset build, preflight, training, or export
  • the exact warning or error message shown on screen
  • whether the problem is reproducible

4. Scope of support

LoRabbit support documentation covers the current public build of the app itself.

It does not grant rights to third-party models, datasets, or documents, and it does not replace the license terms, model cards, or usage restrictions published by their original providers.

한국어

1. 공개 지원 경로

  • 프로젝트 저장소: https://github.com/Facta-Leopard/LoRabbit
  • 지원 이메일: shteosis@gmail.com
  • 이슈 트래커: https://github.com/Facta-Leopard/LoRabbit/issues
  • 디스커션: https://github.com/Facta-Leopard/LoRabbit/discussions
  • 개인정보처리방침: privacy.html

저장소에서 Issues 또는 Discussions가 활성화되어 있다면, 현재 공개 빌드 기준 기본 공개 지원 경로는 해당 채널입니다.

2. 지원 요청 전에 준비하면 좋은 정보

가능하면 아래 정보를 함께 준비해 주세요.

  • compatibility report
  • preflight summary
  • training overview
  • run log
  • macOS 버전
  • 사용 중인 Apple Silicon 기기 모델
  • 선택한 모델 형식 (예: MLX 준비 디렉터리, safetensors 기반 디렉터리, GGUF 소스)
  • 현재 워크스페이스가 기본 워크스페이스인지 사용자 지정 워크스페이스인지

이 정보가 있으면 모델 형식 문제인지, 워크스페이스 접근 문제인지, 데이터셋 전처리 문제인지, 학습 메모리 문제인지 더 빠르게 구분할 수 있습니다.

3. 문의 시 함께 적어주면 좋은 내용

  • 어떤 모델 입력을 선택했는지
  • 어떤 데이터셋 입력을 선택했는지
  • 문제가 모델 검사, 데이터셋 빌드, preflight, 학습, 내보내기 중 어디에서 발생했는지
  • 화면에 표시된 정확한 경고 또는 오류 문구
  • 문제가 다시 재현되는지 여부

4. 지원 범위

LoRabbit 지원 문서는 현재 공개 빌드 기준의 앱 자체 동작을 다룹니다.

이 문서는 제3자 모델, 데이터셋, 문서에 대한 권리를 부여하지 않으며, 원 저작권자나 제공자가 공개한 라이선스, 모델 카드, 사용 제한을 대체하지 않습니다.
